QBASIC Programs
==============

**Author: Matschieu**

# Project description

This projects contains some simple programs written in QBasic during 90's.

# Running the project

* Build Docker image and run it with redirection on port 8080:
```
docker build -t qbasic . && docker run --rm -p 8080:8080 qbasic
```
* Then run the URL `http://localhost:8080/vnc.html` in a web browser and click on connect. You can then use Qbasic to open and run the programs.
